Acerca de Nosotros!

Núcleo, es una organización académica sin fines de lucro que ofrece cursos de programación básica gratuita y personalizado que permite a los estudiantes aprender a su propio ritmo. Nuestra misión es la de proporcionar una educación gratuita de clase mundial para cualquier persona en cualquier lugar.

Nuestros Cursos

Aprende Html

HTML es un lenguaje para modelar la estructurar la información. Mediante una serie de etiquetas podremos describir los contenidos de nuestro sitio web, es decir, el esqueleto de nuestro sitio.

Aprende Css

La tecnología CSS está diseñada para marcar la separación del contenido de las páginas web y la forma de presentación de estas. Lo que genera múltiples beneficios, como: presentar el documento final en diferentes estilos (pantalla, voz, impresión).

Aprende JavaScript

Javascript es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás. (Está bien, no todo, pero es sorprendente lo que puedes lograr con unas pocas líneas de código Javascript.
